# AstrBot 搜图神器插件 (astrbot_plugin_soutushenqi)

🎨 极致稳定的 AI 驱动智能搜图插件，深度融合 **视觉大模型 (VLM) 九宫格淘汰比对** 与 **Bing 图源智能兜底**，彻底告别“搜不到图”的尴尬。

## ✨ 核心特性

### 🖼️ 智能搜图与比对
* **🧠 VLM 裁判优选**：摒弃传统“盲盒搜图”。并发抓取多张候选图片拼接成九宫格，交由视觉大模型仔细端详，精准选出最符合用户需求的那一张。
* **🗣️ 深度自然语言集成**：
    * **问答配图**：当用户提问“歼20是什么？”时，大模型在科普的同时，会自动附上高清配图。
    * **意图识别**：对机器人说“帮我找一张布偶猫的图”，大模型能精准识别意图并自动调用工具，无需死记硬背 `/搜图` 指令。
* **⚙️ 极细粒度控制**：【指令搜图】、【自然语言搜图】、【百科配图】三大场景的 VLM 比对开关相互独立，完全听凭后台调遣。

### 🛡️ 企业级容错架构
* **🚧 强力防盗链穿透**：针对国内主流图床（百度、堆糖、B站等），内置动态 Referer 与多重 Header 伪造机制，极大降低 `HTTP 403/400` 拦截率。
* **🔀 智能图源兜底 (Fallback)**：
    * **百度隔离**：主动屏蔽防盗链重灾区（百度系图床），提高基础存活率。
    * **Bing 无缝混入**：当主图源（搜图神器）存活率低于 30% 时，**瞬间无缝切换并并发抓取 Bing 必应图库**。将残存图与 Bing 补充图混合，一同送交 VLM 淘汰，确保绝不空手而归。
* **⚡ 内存级极速流转**：下载与拼图彻底解耦。VLM 选定后，直接从内存读取 `bytes` 缓存发送，省去二次下载，响应速度翻倍。

## 📦 安装方法

### 前置要求
* AstrBot 4.10+
* Python 3.10+

### 依赖库
插件需要以下依赖（会自动安装）：
* `playwright`：无头浏览器渲染与抓取
* `aiohttp`：高并发异步网络请求
* `Pillow`：图片内存级拼合处理

### 安装步骤
1.  进入 AstrBot 插件目录：
    ```bash
    cd AstrBot/data/plugins/
    ```
2.  克隆本仓库（或从插件市场直接安装）：
    ```bash
    git clone [https://github.com/YourGitHub/astrbot_plugin_soutushenqi](https://github.com/YourGitHub/astrbot_plugin_soutushenqi)
    ```
3.  安装 Playwright 内核（若环境未安装）：
    ```bash
    playwright install chromium
    ```
4.  在 AstrBot 后台重启机器人或重载插件即可。

## 🔧 配置说明 (可视化面板)

插件安装后，可在 AstrBot 后台配置面板进行高度自定义：

| 配置项 | 默认值 | 说明 |
| :--- | :--- | :--- |
| `enable_cmd_vlm_selection` | `true` | **【指令搜图】** (/搜图) 是否启用 VLM 九宫格淘汰比对 |
| `enable_nl_search_vlm_selection` | `true` | **【自然语言搜图】** (如“帮我搜图”) 是否启用 VLM 淘汰比对 |
| `enable_explanation_image` | `true` | **【解释附图】** 是否启用功能（解答明确实体时自动配图） |
| `enable_explanation_vlm_selection` | `false` | **【解释附图】** 此场景下是否启用 VLM 淘汰比对（关则取首图） |
| `vlm_provider_id` | （空） | 用于比对的 VLM 模型提供商 ID。留空则默认使用当前对话的模型 |
| `batch_size` | `9` | 单批次候选图片数量（建议 `4` 或 `9`）。**当主图源存活率 < 30% 时，将触发 Bing 补充机制** |

## 🎮 使用指南

### 1. 指令模式 (经典)
精准、快速地通过命令触发：
* `/搜图 最终幻想14`
* `/搜图 歼20高清壁纸`

### 2. 自然语言模式 (智能)
无需命令，像和朋友聊天一样提出需求：
* *“帮我找一张初音未来的壁纸”*
* *“给我看看长城长什么样”*

### 3. 百科配图模式 (科普)
触发“What is”类问题时，图文并茂：
* **用户**：*“歼35战斗机是什么？介绍一下”*
* **Bot**：*(发送歼35高清图)* + *"歼35是咱家自研的新型隐身战斗机，也就是空军刚官宣不久的五代机，具备卓越的隐身性能......"*
